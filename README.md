Advanced Malicious Application Detection using Deep Learning

Project Overview

This project focuses on building a **web-based system for detecting malicious applications** using deep learning techniques. 
The system allows users to scan files, monitor threats in real time, generate reports, and manage upload history. It aims to enhance 
cybersecurity by providing **accurate, scalable, and user-friendly malicious application detection**.
